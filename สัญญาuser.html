สัญญาฉบับนี้ทำขึ้นระหว่าง นางสาวณัฐรินทร์ สุขใส เรียกว่า “ผู้ให้เช่า” ฝ่ายหนึ่ง กับ ... ถือบัตรประชาชนเลขที่ ... เรียกว่า “ผู้เช่า” อีกฝ่ายหนึ่ง โดยมีรายละเอียดดังนี้

ข้อ 1. รายละเอียดห้องพัก: ผู้ให้เช่าตกลงให้ผู้เช่าเช่าห้องพักเลขที่ ... ชั้นที่ ... ของอาคาร Cozy Home เพื่อพักอาศัย ข้อ 2. ระยะเวลาการเช่า: มีกำหนดระยะเวลา 6 เดือน เริ่มตั้งแต่วันที่ ... ถึงวันที่ ... ข้อ 3. ค่าเช่าและเงินประกัน: ตกลงชำระค่าเช่าเดือนละ ... บาท และวางเงินประกันความเสียหาย ... บาท ซึ่งจะได้รับคืนเมื่อสิ้นสุดสัญญาตามเงื่อนไข ข้อ 4. ค่าบริการและสาธารณูปโภค: ค่าไฟฟ้าหน่วยละ ... บาท ค่าน้ำหน่วยละ ... บาท และค่าปรับชำระล่าช้าวันละ ... บาท ข้อ 5. การชำระเงิน: ผู้เช่าตกลงชำระค่าเช่าและค่าบริการต่างๆ ภายในวันที่ ... ของทุกเดือน
